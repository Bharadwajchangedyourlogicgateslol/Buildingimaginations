# Hand Control

Control your mouse using just your hand and webcam. No mouse needed.

Built with MediaPipe, OpenCV and Python. Simple, lightweight, and actually usable.

*DISCLAIMER: I hold credits only about the idea, the rest of the code was written by a local offline coding llm*

## How to use 
-Point your index finger towards the camera to move the cursor.
 
-Bring thumb and index finger close = left click.
 
-Hold them together for 4 seconds = drag mode.
 
-Thumb + middle finger = right click.
 
-Make a fist and move hand up/down = scroll.
 
-Press ESC key to quit.
 
## How to Install

1. Make sure you have **Python 3.10 or 3.11** installed.

2. Download or clone this repo and extract it.

3. Open Command Prompt / Terminal **inside the project folder** and run:

 ```bash
  pip install opencv-python numpy pyautogui keyboard pygame ttkbootstrap
  pip install mediapipe
````
## Features

- Move cursor with your index finger
- Left click → Quick pinch (thumb + index finger)
- Drag → Hold pinch for 4 seconds
- Right click → Pinch thumb + middle finger
- Scroll → Make a fist and move hand up/down
- Smooth cursor movement
- Live settings panel (sensitivity, click thresholds, etc.)
