## It is expected that mideapipe or pygame library would not install properly on its own 
1.For Pygame:

 Go to https://www.pygame.org/downloads.shtml
 
 Download the latest stable version for your Python (.whl file or source).
 
 Put the downloaded file inside your project folder.
 
 Open command prompt inside your project folder and run:Bashpip install pygame-*.whl

2.For MediaPipe:

 Go to https://pypi.org/project/mediapipe/#files
 
 Download the .whl file that matches your system (for example mediapipe-0.10.35-py3-none-win_amd64.whl for normal Windows 64-bit).
 
 Put that .whl file directly in your project folder.
 
 Open command prompt in the project folder and run:Bashpip install mediapipe-*.whl
